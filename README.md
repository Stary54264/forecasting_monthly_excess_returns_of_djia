# Forecasting Monthly Excess Returns Of DJIA

## Overview

This repo contains a study that builds a time series model using historical data to predict the futrue excess returns of DJIA.


## File Structure

The repo is structured as:

-   `data` contains the data obtained from Yahoo Finance.
-   `report` contains the files used to generate the report, including the Quarto document and the PDF of the report. 
-   `script` contains the R script used to download and clean data.
